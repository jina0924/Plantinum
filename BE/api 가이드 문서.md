### OTP 생성

- 기기와 연결되지 않았고 OTP도 없는 상태에서 OTP 발급
- 5분 후 자동으로 삭제 (blank)

- 로그인 사용자 - 토큰 사용
- POST

- URL

```
http://127.0.0.1:8000/api/v1/plants/myplant/{물주기 식물 pk}/otp/
```

- Request Parameters

| Name           | Type | Description             | Mandatory | Example |
| -------------- | ---- | ----------------------- | --------- | ------- |
| 물주기 식물 pk | Int  | 물주기 등록한 식물의 pk | O         | 1       |

- Response

```
{
    "otp_code": "636631"
}
```

```
{
    "result": "이미 발급되었거나 연결되었습니다."
}
```
